#MenteCart Backend

## Tech Stack
- Node.js
- Express
- TypeScript
- MongoDB
- JWT

#  Features
- Authentication
- Service Management
- Cart System
- Booking System
- Slot Capacity Protection

npm install

## Run Development Server

npm run dev

PORT=
MONGO_URI=

## API Endpoints

POST /api/auth/signup
POST /api/auth/login
GET /api/Services
POST /api/bookings/checkout

## Business Logic
- Atomic slot capacity management
- Duplicate cart prevention
- Booking lifecycle management
